# AntiTheft APK Hosting Site

This is a premium static website designed to host and serve the AntiTheft APK file.

## Features
- **Stunning Design**: Modern dark-mode UI with glassmorphism and smooth animations.
- **Mobile Responsive**: Works perfectly on all devices.
- **SEO Optimized**: Includes meta tags for search engines and social sharing.
- **Render Ready**: Includes `render.yaml` for instant deployment.

## Deployment to Render
1. Create a new repository on GitHub and push this code to it.
2. Log in to your [Render dashboard](https://dashboard.render.com).
3. Click **New +** and select **Blueprint**.
4. Connect your GitHub repository.
5. Render will automatically detect the `render.yaml` file and set up your static site.

## File Structure
- `index.html`: Main landing page.
- `public/css/style.css`: Custom styling.
- `public/downloads/app-debug.apk`: The APK file for users to download.
- `render.yaml`: Configuration for Render deployment.
